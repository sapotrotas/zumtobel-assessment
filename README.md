# Zumtobel assessment

[Deployed here](https://zumtobel-assessment.nuxt.dev/)

## Features
- Nuxt with SSR
- Vitest
- Tailwindcss
- NuxtHub
- Nuxt Security
- Cache API requests
- Input methods: 
  - Advent of Code user session 
  - Static files (./public/challenges)

## Setup

Install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev
```
## Unit Tests

```bash
# npm
npm run test
```

## Lint

```bash
# npm
npm run lint
```
