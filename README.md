# Zumtobel assessment

## Features
-- SSR
-- NuxtHub
-- Nuxt Security
-- Tailwindcss
-- Cache API requests

-- Input methods: 
--- Advent of Code session 
--- Static files (./public/challenges)

## Setup

Install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev
```
